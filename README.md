# CPP3_BrickGame_v2.0 — Snake Game (MVC + FSM + Qt)

## Описание

**CPP3_BrickGame_v2.0** — реализация классической игры «Змейка» на языке C++, выполненная в парадигме объектно-ориентированного программирования с использованием:
- архитектурного паттерна MVC
- конечных автоматов (FSM)
- модульного тестирования
- GUI-интерфейса на Qt
  
Проект демонстрирует чистое архитектурное разделение логики и интерфейса, высокое качество кода и удобную систему сборки.
Это улучшенная и расширенная версия предыдущего проекта **C7_BrickGame_v1.0**.

## Возможности

- Классическая механика «Змейки»
- Управление направлением движения
- Подсчёт очков
- Хранение рекордов
- Корректная обработка столкновений
- Плавная работа конечного автомата состояний
- Два интерфейса: консольный (ncurses), графический (Qt)

## Используемые технологии

- C++20
- Qt 5/6 — GUI
- Google Test
- Make / Makefile
- ncurses
- ООП + MVC
- Finite State Machine
- Google Style Guide
- Git + GitHub


## Запуск

1. **Клонирование**
   ```bash
    git clone https://github.com/IAmAnAlligator/CPP3_BrickGame_v2.0.git
    cd CPP3_BrickGame_v2.0
   ```

3. **Сборка**
   ```bash
   make install
   ```
